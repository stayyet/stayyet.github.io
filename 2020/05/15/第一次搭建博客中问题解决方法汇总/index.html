第一次搭建博客中问题解决方法汇总

ways
发布日期:   2020-05-15

博客搭建问题汇总按时间顺序呈现

nodejs调试及确认成功方法

  • nodejs
    官网下载即可，本人速度还行
  • Git
    建议在淘宝镜像源下载最新版
    nodejs下载完成后创建一个文件夹，我放在我的D盘里，因为D盘比较大，我专门用来放游戏的，如果是系统盘512G或者1T的土豪当我没说
    在这个文件夹中下载[hexo-script]的安装脚本
    打开git bash然后执行命令行curl -O https://cdn.jsdelivr.net/gh/kjhuanhao/hexo-script@master/install.sh

在存放install.sh脚本的目录中,右键打开git bash然后运行以下命令:
source install.sh test_node

如果出现了上图，恭喜啦！nodejs百分之99已经OK了，不需要你捣鼓了。

Git调试和确认成功方法

Git安装方法超详细
如果你想要检测git环境或者查看当前git的版本号,您可以在存放install.sh脚本的目录中,右键打开git bash运行以下命令:
source install.sh test_git

如果出现了上图，也恭喜啦！Git百分之99已经OK了，也不需要你捣鼓了。

安装hexo博客

您需要在你创建的博客目录也就是存放install.sh脚本的目录中,右键打开git bash运行以下命令:
source install.sh hexo_win
此时不要急，等待下图的出现

当您看到代码最后一行出现Please run hexo s to check it out!的提示,证明您已经安装成功,此时你可以运行hexo s 然后查看你的博客
先运行命令cd hexoblog
紧接着运行命令hexo s

可以看到出现了一段地址:http://localhost:4000
你直接点我这个就能进入你的本地博客了

生成SSH秘钥

关于Github的注册及库的创建比较基础，不做说明
为了撰写博客的库（repository）有格式要求：你的Github账户名.github.io
Windows环境下生成SSH key且连接GitHub
这个教程有windows的教程及检验成功方法

部署hexo博客

部署博客必要的配置，首先完成这个
首先你需要打开根目录配置文件_config.yml

找到最后一行,按照以下的格式修改即可

你只需要修改repo后面的信息即可
其实https的也行
您需要在你的博客目录也就是存放install.sh脚本的目录中,右键打开git bash运行以下命令:
source install.sh deploy


你需要将up.sh复制到hexoblog文件,也就是复制到你的博客根目录下

然后在存放up.sh脚本的目录中,右键打开git bash运行以下命令:
source up.sh
之后即可自动部署到你的github或者coding上

最后一步报错提示

通过hexo d部署时报Error: Spawn failed错误:
通过Github点击右上角头像，菜单栏倒数第二的setting，点右边栏目中的email,去掉Keep my email addresses private前面的勾选
